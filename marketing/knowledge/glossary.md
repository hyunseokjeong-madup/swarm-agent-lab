# 마케팅 지표 용어집 (Glossary)

쓸수록 자라는 지표 사전. 정의·공식·주의. (계산은 항상 코드로 검산 → `reconcile.py`/`summarize.py`)

| 용어 | 정의 / 공식 | 주의 |
|------|-------------|------|
| Impressions | 광고 노출 수 | 뷰어블(viewable)과 구분 |
| Reach | 도달한 순 사용자 수 | reach ≤ impressions |
| Frequency | impressions / reach | 캡 ≈2~3, 초과 시 피로 |
| CTR | clicks / impressions | outbound vs all 구분 |
| CPC | spend / clicks | — |
| CPM | spend / impressions × 1000 | ×1000 |
| CVR | conversions / clicks | /impressions 정의도 존재 |
| CPA / CPI | spend / conversions(installs) | — |
| ROAS | revenue / spend | 배수(x) 표기 |
| POAS | revenue × margin / spend | 수익성 ROAS, <1이면 손실 |
| ROI | (revenue − spend) / spend | % |
| AOV | revenue / orders | — |
| LTV | 고객 생애 가치(코호트 누적 또는 ARPU×수명×마진) | 정의 문서화 |
| CAC | 고객획득비용 | LTV/CAC ≥ 3 권장 |
| Payback | CAC / 월 공헌 | 개월 |
| Retention Dn | day-n 잔존 / 설치 | 코호트 |
| eCPM | revenue / impressions × 1000 | 퍼블리셔 측 |
| VTR | views / impressions | 동영상 |
| Fill rate | served / requested | 인벤토리 |
| IS(노출점유율) | 노출 / 가능노출 | 손실 IS(예산/순위) |
| Blended ROAS | Σrevenue / Σspend | 가중(단순평균 금지) |
| Incrementality | 광고로 인한 순증분(holdout 대비) | 플랫폼 전환과 다름 |
